La ligne de commande

Gestion de documents

Pour créer un dossier je dois utiliser cette commande : mkdir

Pour lister les fichiers, les sous dossiers, en gros le contenu du repertoire dans lequel je me trouve : ls

Pour afficher le manuel des commandes : man ls/

Pour ouvrir un fichier html et le visualiser dans un navigateur : x-www-browser index.html ou firefox index.html

Pour afficher le chemin du dossier courant : pwd

Pour ouvrir un fichier, par exemple index.html : nano index.html

Pour se déplacer dans les dossiers : cd Documents (par exemple)

Pour installer un programme préalablement téléchargé sur le Web : sudo dpkg -i programme.deb

Pour créer un fichier : touch ou echo exemple : touch index.html

Privilèges Administrateur / Super Utilisateur

Pour entrer en mode Super Utilisateur :

  • sudo -s
  • sudo -i
  • sudo

Mise à jour / installation de paquets

Pour mettre à jour la liste des paquets disponibles : sudo apt update

Pour mettre à jour les paquets déjà installés : sudo apt upgrade

Pour installer un logiciel spécifique : sudo apt install logiciel spécifique

Pour rechercher un texte dans les descriptions de paquet : apt search terme à rechercher
